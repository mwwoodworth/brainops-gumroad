# Roofing Estimation Intelligence (GR-ROOFINT) - Start Here

Thank you for purchasing the Roofing Estimation Intelligence pack. This toolkit is designed to help roofing contractors and project managers analyze bids, calculate costs, and assess risks using AI.

## What's Included
1. **Roofing-Intelligence-Guide.md**: Comprehensive manual on using these prompts (PDF can be generated from this file).
2. **Prompts Library**:
   - `pre-bid-analysis.txt`: Analyze project requirements before bidding.
   - `cost-calculation.txt`: AI-assisted material and labor cost estimation.
   - `risk-assessment.txt`: Identify potential project pitfalls.
3. **ROI-tracker.csv**: Track the profitability of your winning bids.
4. **Sample Output**: See what a good AI analysis looks like.

## Quick Start
1. Open `Roofing-Intelligence-Guide.md` to understand the workflow.
2. Select a prompt from the `prompts/` folder.
3. Paste the prompt into your LLM (ChatGPT, Claude, etc.) followed by your project notes or raw bid data.
4. Review the output and adjust your final bid in the `ROI-tracker.csv`.
