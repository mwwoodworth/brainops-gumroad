# Modern Command Center UI Kit

**Version:** 1.0
**Stack:** Next.js 14, Tailwind CSS, Lucide Icons

## Overview
Stop building boring admin panels. This UI Kit gives you the exact layout, components, and styling used in the "BrainOps Command Center". It's dark-mode native, futuristic, and data-dense.

## What's Included
1.  **Dashboard Layout**: The responsive sidebar + header layout.
2.  **Metric Cards**: Beautiful glassmorphic cards for displaying stats.
3.  **Task Stream**: A "Live Feed" component for activities.
4.  **Income Lab**: The complete "Product Management" interface code.

## Installation
1.  Copy the `components/` folder to your Next.js project.
2.  Install dependencies: `npm install lucide-react framer-motion clsx tailwind-merge`
3.  Import `DashboardLayout` in your `layout.tsx`.

## Customization
- **Colors**: The kit uses standard Tailwind colors (Cyan, Emerald, Purple). customize in `tailwind.config.ts`.
- **Animations**: Includes `framer-motion` variants for smooth entry animations.

## Bonus
Includes the "Gumroad Product Launch Pad" logic for tracking digital asset builds.
