# PM Accelerator (GR-PMACC) - Start Here

Welcome to the PM Accelerator. This kit is designed to cut your project management overhead by 50% using targeted AI prompts and templates.

## What's Inside
- **PM-Accelerator-Playbook.md**: Your operating manual (exportable to PDF if needed).
- **Prompts**:
  - `project-intelligence.txt`: Summarize docs and extract action items.
  - `resource-optimization.txt`: Balance workloads.
  - `stakeholder-alignment.txt`: Draft perfect updates.
- **Templates**:
  - `templates/weekly-status.md`
  - `templates/meeting-recap.md`
- **Video Walkthrough**: Script in `video/pm-accelerator-walkthrough.txt`.

## Getting Started
1. Read the Playbook.
2. Duplicate the templates to your note-taking app (Notion, Obsidian, or Docs).
3. Use the `project-intelligence.txt` prompt to process your next meeting transcript or project brief.
