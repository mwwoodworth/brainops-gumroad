# Content Pipeline Automation (GR-CONTENT) - Start Here

Streamline your content creation from ideation to social posting using AI and Make.com.

## Included
- **Blueprints**: `content-pipeline-main.json` (RSS/Idea -> AI -> Draft).
- **Prompts**: `prompts/content-repurpose-library.txt` (Turn 1 blog post into 10 tweets).
- **Templates**: Calendar CSV and Notion structure.
- **Guide**: `Content-Pipeline-Setup.md`.

## Setup Overview
1. Set up your Content Calendar (Notion or Sheets).
2. Import the Make.com blueprint.
3. Configure the AI module to read your style guide.
4. Watch drafts appear automatically in your "Review" folder.
