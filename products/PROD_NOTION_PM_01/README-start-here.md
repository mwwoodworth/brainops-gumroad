# Notion PM Command Center (GR-PMCMD) - Start Here

The ultimate Notion operating system for project managers.

## Setup
1. Open `notion-template-link.txt`.
2. Click the link and "Duplicate" to your workspace.
3. Watch `video/pm-command-center-walkthrough.txt` (script) to learn the flows.
4. Read `Notion-PM-Command-Center-Guide.md` for advanced usage.

## Features
- Projects & Tasks DB
- Meeting Notes (Linked to Projects)
- CRM / Stakeholder Directory
- Resource Management View
