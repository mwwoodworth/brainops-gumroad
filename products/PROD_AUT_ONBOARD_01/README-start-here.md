# Make.com Onboarding Automation (GR-ONBOARD) - Start Here

Automate your client or employee onboarding process using Make.com.

## Contents
- **Blueprints**: Importable JSON files for Make.com.
  - `onboarding-main.json`: The core logic.
- **Guides**: Setup and troubleshooting instructions.
- **References**: Module documentation.
- **Video**: `video/onboarding-setup.txt` (Script).

## Prerequisites
- A Make.com account (Free tier works for low volume, Starter recommended).
- Google Sheets (or Airtable).
- Slack/Email (for notifications).
- OpenAI API Key (optional, for welcome customization).

## Quick Install
1. Log into Make.com.
2. Create a new scenario.
3. Click "More" -> "Import Blueprint".
4. Select `blueprints/onboarding-main.json`.
5. Connect your accounts as detailed in `Onboarding-System-Setup.md`.
