# Troubleshooting

## Common Errors

### 401 Unauthorized
- **Cause**: API Key or connection expired.
- **Fix**: Reauthorize the connection in the module settings.

### Bundle Error / Missing Data
- **Cause**: The trigger sheet row is empty or missing columns.
- **Fix**: Ensure your Google Sheet headers match exactly what the scenario expects.

### Gmail: "Application-specific password required"
- **Fix**: If using personal Gmail, you must enable 2FA and generate an App Password for Make.com.
